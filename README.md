In this phase of the project, advanced DML and DDL techniques were applied to effectively normalize the dataset and improve data integrity. The normalization process followed Third Normal Form (3NF) principles and included the following steps:
Identification of Redundant Data: Analyzed existing tables to detect redundant or duplicated information that could lead to inconsistencies.
Application of 3NF Principles: Evaluated table structures to ensure that all attributes were functionally dependent on primary keys and free of transitive dependencies.
Elimination of Redundant Columns: Restructured non-compliant tables by removing unnecessary columns and separating data into appropriately related tables.
This process enhanced data consistency, reduced redundancy, and created a more efficient and scalable database structure for analysis
